# 幹話產生器 Dry Talk
### 選擇四種職業中,產生出你對那職業想說的幹話!

## 開發環境與套件
+ Node.js -v10.15.0 
+ Express -4.17.1 
+ Express-Handlebars -5.2.1 
+ Nodemon -2.0.7
+ body-parser -1.19.0

## 安裝與執行要點
 1. 在本地端建立專案資料夾 .
 ```
    mkdir drytalk_generator
 ```     
 2. 將專案下載到專案資料夾並進入資料夾.
 ```
    cd drytalk_generator
      
 3. 安裝npm.
 ```
       npm install
 ```
 4. 使用nodemon開啓專案.
 ```
       nodemon app.js  
 ```   
 5. 觀察終端機輸出的訊息,是否成功開啓伺服器.
 ```
      The Express server is Run successfully URL http://localhost:3000
      看到終端機出現上面字樣表示開機成功
